## 技术栈建议

| 层级 | 推荐方案 | 理由 |
|------|----------|------|
| 前端框架 | 现有 HTML/JS 原型 → 视情况迁移 React | 原型已完成，短期可直接用 |
| 后端 / 数据库 | Supabase | PostgreSQL + Auth + Storage + Realtime，一套全搞定 |
| 支付 | Stripe（国际）+ 微信支付（国内） | 成熟 SDK，接入半天 |
| 邮件 | Resend 或 SendGrid | 简单易用，免费额度够用 |
| 部署 | Vercel + Supabase | 免运维，一键部署 |
| 监控 | Sentry | 免费额度够，错误自动报警 |
